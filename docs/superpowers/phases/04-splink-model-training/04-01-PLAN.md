<task type="auto">
  <name>Task 1: Create Splink training script</name>
  <files>scripts/train_splink_model.py</files>
  <action>
  Create `scripts/train_splink_model.py` that:

  1. Import `_connect_ducklake` from `dagster_pipeline.defs.name_index_asset` (add the project src to sys.path if needed for script execution).

  2. Connect to DuckLake and read a 2M stratified sample from `lake.federal.name_index`:
     ```sql
     CREATE OR REPLACE TABLE name_index_sample AS
     SELECT * FROM (
         SELECT *, ROW_NUMBER() OVER (PARTITION BY source_table ORDER BY random()) as rn
         FROM lake.federal.name_index
     )
     WHERE rn <= 50000
     ```
     (50K per source table × ~44 tables ≈ 2.2M sample — adjust cap to get ~2M total)

     IMPORTANT: Read the sample into a LOCAL DuckDB table (not in the lake). The lake connection is for reading; Splink needs a local table to work with. Pattern:
     ```python
     conn = _connect_ducklake()
     # Read sample from lake into local table
     conn.execute("""
         CREATE TABLE name_index_sample AS
         SELECT * FROM (
             SELECT *, ROW_NUMBER() OVER (PARTITION BY source_table ORDER BY random()) as rn
             FROM lake.federal.name_index
         ) WHERE rn <= 50000
     """)
     # Drop the rn column
     conn.execute("ALTER TABLE name_index_sample DROP COLUMN rn")
     ```

  3. Configure Splink settings (same as PoC, proven to work):
     ```python
     settings = SettingsCreator(
         link_type="dedupe_only",
         unique_id_column_name="unique_id",
         blocking_rules_to_generate_predictions=[
             block_on("last_name", "first_name"),
             block_on("last_name", "zip"),
         ],
         comparisons=[
             cl.NameComparison("first_name"),
             cl.NameComparison("last_name"),
             cl.ExactMatch("city").configure(term_frequency_adjustments=True),
             cl.ExactMatch("zip").configure(term_frequency_adjustments=True),
         ],
         max_iterations=10,
         em_convergence=0.001,
     )
     ```

  4. Train the model:
     ```python
     linker = Linker("name_index_sample", settings, db_api=DuckDBAPI(connection=conn))
     linker._settings_obj._probability_two_random_records_match = 1e-6
     linker.training.estimate_u_using_random_sampling(max_pairs=5_000_000, seed=42)
     linker.training.estimate_parameters_using_expectation_maximisation(
         block_on("last_name", "first_name"), fix_u_probabilities=False
     )
     linker.training.estimate_parameters_using_expectation_maximisation(
         block_on("last_name", "zip"), fix_u_probabilities=False
     )
     ```

  5. Save the trained model:
     ```python
     import os
     os.makedirs("models", exist_ok=True)
     linker.misc.save_model_to_json("models/splink_model.json", overwrite=True)
     ```

  6. Run a quick validation prediction on the sample:
     ```python
     results = linker.inference.predict(threshold_match_probability=0.8)
     clusters = linker.clustering.cluster_pairwise_predictions_at_threshold(
         results, threshold_match_probability=0.85
     )
     ```
     Log: sample count, cluster count, multi-record match count.

  7. Print summary stats:
     - Sample size
     - Training time
     - Model file size
     - Cluster count from validation
     - Multi-record matches (clusters with 2+ records)

  The script should be runnable standalone:
  ```bash
  cd ~/Desktop/dagster-pipeline
  python scripts/train_splink_model.py
  ```
