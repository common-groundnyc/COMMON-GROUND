The current query builds pol_entities from a CTE with UNION ALL of donors, candidates, vendors, lobbyists, clients. Add new CTEs:

```sql
-- Add to the WITH clause before all_entities:

expenditure_payees AS (
    SELECT UPPER(TRIM(payee_name)) AS entity_name, 'PAYEE' AS entity_type,
           SUM(TRY_CAST(amount AS DOUBLE)) AS total_amount,
           COUNT(*) AS tx_count
    FROM lake.city_government.nys_campaign_expenditures
    WHERE payee_name IS NOT NULL AND LENGTH(payee_name) > 3
    GROUP BY UPPER(TRIM(payee_name))
    HAVING COUNT(*) >= 2
),
offyear_donors AS (
    SELECT UPPER(TRIM(name)) AS entity_name, 'DONOR' AS entity_type,
           SUM(TRY_CAST(amount AS DOUBLE)) AS total_amount,
           COUNT(*) AS tx_count
    FROM lake.city_government.cfb_offyear_contributions
    WHERE name IS NOT NULL AND LENGTH(name) > 3
    GROUP BY UPPER(TRIM(name))
),
doing_business_donors AS (
    SELECT UPPER(TRIM(name)) AS entity_name, 'DOING_BUSINESS' AS entity_type,
           SUM(TRY_CAST(amount AS DOUBLE)) AS total_amount,
           COUNT(*) AS tx_count
    FROM lake.city_government.doing_business_contributions
    WHERE name IS NOT NULL AND LENGTH(name) > 3
    GROUP BY UPPER(TRIM(name))
),
```

And add to the all_entities UNION:
```sql
SELECT * FROM expenditure_payees UNION ALL

```

**For citywide payroll**: This is 6.8M rows but the value is employee→agency connections. Create a SEPARATE table rather than bloating pol_entities:

```sql
CREATE OR REPLACE TABLE main.graph_payroll_agencies AS
SELECT UPPER(TRIM(agency_name)) AS agency_name,
       COUNT(DISTINCT UPPER(TRIM(first_name || ' ' || last_name))) AS employee_count,
       SUM(TRY_CAST(base_salary AS DOUBLE)) AS total_payroll,
       COUNT(*) AS record_count
FROM lake.city_government.citywide_payroll
WHERE agency_name IS NOT NULL
GROUP BY UPPER(TRIM(agency_name))
```

This gives agency-level stats that pay_to_play can cross-reference (which agencies are donors connected to?).

**Also add expenditure edges** — who politicians pay:
```sql
CREATE OR REPLACE TABLE main.graph_pol_expenditures AS
SELECT UPPER(TRIM(filer_name)) AS filer_name,
       UPPER(TRIM(payee_name)) AS payee_name,
       SUM(TRY_CAST(amount AS DOUBLE)) AS total_spent,
       COUNT(*) AS payment_count,
       MAX(date) AS latest_date,
       FIRST(purpose) AS purpose
FROM lake.city_government.nys_campaign_expenditures
WHERE filer_name IS NOT NULL AND payee_name IS NOT NULL
  AND LENGTH(payee_name) > 3
GROUP BY UPPER(TRIM(filer_name)), UPPER(TRIM(payee_name))
```
